--- layout: post title: vi编辑器的属性设置 categories: - Linux - OS tags: - HOWTO - Linux - Note - Vim published: true comments: true --- 在当前用户目录下用vi新建并打开.vimrc文件 vi ~/.vimrc 
" 在文件中写入以下设置

" 设置显示行号
set nu

" 设置tab的长度为4
set tabstop=4 

" 打开状态栏标尺
set ruler

" 突出当前行
set cursorline

" 自动语法高亮
syntax on
保存退出后即可生效
